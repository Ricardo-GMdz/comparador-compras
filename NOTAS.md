-quisiera crear un proyecto donde tenga un agente que pueda buscar información sobre productos que le diga, y pueda buscar en internet la información y poder comprar diferentes opciones de compra. Osea este agente funcionara para validar diferentes proveedores a quienes puedo contactar o validar información sobre el producto que este buscando. Debe ser capaz de darme información sobre varios productos del mismo que este buscando, validar y hacer comparaciones de precios entre proveedores y designar cual seria la mejor opción para la compra. En dado caso que este buscando algún producto el cual tenga una versión mejor y entre en un rango de precio similar o competitivo también puedes mencionarlo. El proyecto aun no tiene nombre, asi que puedes designarle uno.
