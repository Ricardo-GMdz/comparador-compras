# v2.2 — Análisis por proveedor (detalle, catálogo, favoritos, orden) — Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Sumar precio de catálogo y dirección al dato del proveedor, una vista de detalle al clickear el nombre, favoritos múltiples, y orden/filtro por precio y favoritos.

**Architecture:** Extiende los tipos y contratos existentes sin romperlos. `catalogPrice`/`address` son datos del sourcing (opcionales, defensivos); `favorite` es gestión manual (como `status`/`notes`: el sourcing nunca lo pisa, no se publica). El frontend suma un modal de detalle y una barra de orden client-side. Spec: `docs/superpowers/specs/2026-07-12-v2.2-analisis-detalle-favoritos-design.md`.

**Tech Stack:** el existente (TS/Node ESM, Hono, zod, vitest, vanilla JS). Sin dependencias nuevas.

**Convenciones para TODAS las tareas:** TDD (test primero → correr y ver FALLAR → implementar → correr y ver PASAR → commit). Comandos con prefijo `rtk`. Comentarios en español, identificadores en inglés. Inmutabilidad. Antes de cada commit con código: `rtk pnpm run typecheck && rtk pnpm run lint && rtk pnpm test` en verde. Rama: `feat/v2.2-analisis-detalle` (ya creada).

// Precio de catálogo (lista) con moneda; "—" si no hay.
function catalogPrice(s) {
  if (s.catalogPrice === undefined) return "—";
  const currency = s.currency ? " " + esc(s.currency) : "";
  const unit = s.priceUnit && s.priceUnit !== "unknown" ? ` / ${esc(s.priceUnit)}` : "";
  return `$${esc(s.catalogPrice)}${currency}${unit}`;
}

// Precio numérico efectivo para ordenar: mayoreo si hay, si no catálogo; Infinity si ninguno.
function precioEfectivo(s) {
  if (typeof s.wholesalePrice === "number") return s.wholesalePrice;
  if (typeof s.catalogPrice === "number") return s.catalogPrice;
  return Infinity;
}
```

// Filtro client-side por texto, estado y favoritos; luego ordena.
function filtrados() {
  const texto = $("filtro").value.trim().toLowerCase();
  const estado = $("filtroEstado").value;
  const soloFav = $("soloFav").checked;
  const orden = $("orden").value;
  const lista = directorio.filter((s) => {
    const matchTexto =
      texto === "" ||
      s.name.toLowerCase().includes(texto) ||
      s.material.toLowerCase().includes(texto);
    const matchEstado = estado === "todos" || s.status === estado;
    const matchFav = !soloFav || s.favorite === true;
    return matchTexto && matchEstado && matchFav;
  });
  const ordenada = [...lista];
  if (orden === "precio") {
    ordenada.sort((a, b) => precioEfectivo(a) - precioEfectivo(b));
  } else if (orden === "favoritos") {
    ordenada.sort((a, b) => (b.favorite ? 1 : 0) - (a.favorite ? 1 : 0));
  } else if (orden === "nombre") {
    ordenada.sort((a, b) => a.name.localeCompare(b.name));
  } else {
    // "reciente": por lastSeen descendente.
    ordenada.sort((a, b) => (b.lastSeen ?? "").localeCompare(a.lastSeen ?? ""));
  }
  return ordenada;
}
```
